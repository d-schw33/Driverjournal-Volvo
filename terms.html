Application: PoC Driver Journal & Outlook
Last updated: March 2026

1. About this application

This is a personal proof-of-concept application that retrieves vehicle trip data from the Volvo Cars Connected Vehicle API and calendar data from Microsoft Outlook to classify trips as work-related or private. The application is intended for personal use only.

2. Data usage

All data retrieved through the Volvo Cars API — including vehicle trip information, location data, and vehicle identifiers — is processed locally on the user's own device. No data is transmitted to any third-party server or stored outside the user's local environment.
